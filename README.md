## 🎯 Le concept

**Marathon Coran** est un projet éducatif open-source qui permet aux enfants d'apprendre le vocabulaire du Coran en décryptant les **racines arabes** à l'origine de chaque mot. L'objectif : atteindre **88 % de compréhension** du Coran en progressant à travers **5 paliers** de difficulté, couvrant les **49 dernières sourates** du Coran (de la sourate 25 à la sourate 114, soit le juz 30 et une grande partie des juz 29 à 27).

Chaque mini-cours dure **5 à 8 minutes** et suit un format simple et ludique :

1. 📖 **Lire le verset** — le texte arabe (uthmani) avec sa traduction en français (Hamidullah)
2. 🔍 **Décrypter les mots** — 5 à 6 mots analysés avec leur racine, leur sens, un fait amusant et un mnémonique pour les enfants
3. 🧠 **Passer le quiz** — un mini-quiz de 3 questions pour vérifier la compréhension (vocabulaire, racines, sens global)
4. ⭐ **Gagner des XP** — des points d'expérience récompensent la progression

## 🎨 Conçu pour les enfants

Le design est spécialement adapté aux enfants de 7 à 12 ans :

- **Police Nunito** — une fonte arrondie et lisible pour le texte français
- **Police Amiri** — une fonte naskh élégante pour le texte arabe uthmani
- **Couleurs cycliques** — palette de 6 couleurs distinctes pour les familles de racines (rouge, turquoise, or, violet, orange, vert), permettant de repérer visuellement les racines récurrentes
- **Emojis** partout pour rendre l'apprentissage fun et visuel
- **Fun Facts** — chaque mot est accompagné d'un fait amusant de 3 à 5 phrases, avec mise en forme `<em>` et `<strong>` pour les mots importants
- **Gamification** — XP, durée estimée par leçon, badges de complétion
- **Responsive** — fonctionne sur téléphone, tablette et ordinateur
- **Navigation fluide** — boutons précédent/suivant entre les leçons, retour au sommaire du palier

## 🚀 Installation locale

Aucune dépendance requise — c'est du HTML/CSS/JS pur.

1. Clone ce dépôt :
   ```bash
   git clone https://github.com/TON-USER/marathon-coran.git
   cd marathon-coran
   ```

2. Ouvre `index.html` dans ton navigateur, ou lance un serveur local :
   ```bash
   # Python
   python -m http.server 8000

   # Node.js
   npx serve .
   ```

3. Rends-toi sur `http://localhost:8000`

---

## 🌐 Déploiement sur GitHub Pages

1. Crée un dépôt sur GitHub
2. Pousse les fichiers du dossier `marathon-coran-site/` à la racine du dépôt
3. Dans **Settings > Pages**, sélectionne la branche `main` et le dossier `/ (root)`
4. Ton site sera disponible sur `https://TON-USER.github.io/marathon-coran/`
