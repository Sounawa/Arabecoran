🌱 Leçon 4 : Al-Fatiha 1:1-2

⏱️ 8 min • Sourate : Al-Fatiha (1:1-2) • 📜 Lecture Warsh selon Nafi'
🕌 Lecture Warsh selon Nafi' : Ces versets sont identiques dans les deux lectures principales (Hafs et Warsh).
✅ Texte vérifié : King Fahd Quran Complex (Édition Warsh officielle 1445 H)

بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ • الْحَمْدُ لِلَّهِ رَبِّ الْعَالَمِينَ

📜 ÉDITION WARSH SELON NAFI' • POLICE SHEHERAZADE NEW
🌙 Note éthique (Expert Coran) :
"Al-Fatiha" = "L'Ouverture". Cette sourate est la mère du Coran — récitée dans chaque prière.
✅ Source : Tafsir Ibn Kathir, vol.1 p.32 — "La plus grande sourate du Coran"

🔑 Les 7 nouvelles racines (total : 25/50)

Racine arabe Mot dans le verset 🖐️ Geste 💡 Astuce + Source vérifiée
ب س م بِسْمِ 🤲 "Bismi" = "Au nom de..."
بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ
🤲 Geste : Mains jointes devant le cœur (comme pour invoquer)
💡 Astuce : "Bismi" = "Au nom de" → comme "baptême" qui commence "au nom de" !
📚 Source : Quranic Arabic Corpus — racine ب-س-م (vérifiée dans v.1)
ل هـ اللَّهِ ✨ "Allâhi" = "Allah" (Nom propre de Dieu)
بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ
✨ Geste : Mains qui s'ouvrent vers le ciel avec émerveillement
💡 Astuce : Deux "ل" collés : لله → "Al-Lah" = Allah
📚 Source : Tafsir Ibn Kathir — Nom divin unique (vérifié dans v.1)
ر ح م الرَّحْمَٰنِ 💖 "Ar-Rahmâni" = "Le Très Miséricordieux" (attribut divin)
اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ
💖 Geste : Main sur le cœur avec un grand sourire (amour infini)
💡 Astuce : "Rahmân" = "Miséricordieux" → comme "rahma" (miséricorde) en arabe !
📚 Source : Quranic Arabic Corpus — racine ر-ح-م (vérifiée dans v.1)
ر ح م الرَّحِيمِ 🤗 "Ar-Rahîmi" = "Le Très Clément" (attribut divin)
الرَّحْمَٰنِ الرَّحِيمِ
🤗 Geste : Bras ouverts pour faire un câlin (accueil chaleureux)
💡 Astuce : "Rahîm" = "Clément" → comme "rahma" mais avec une douceur continue !
📚 Source : Tafsir Al-Jalalayn — attribut de miséricorde constante (vérifié dans v.1)
ح م د الْحَمْدُ 🙏 "Al-hamdu" = "La louange" (reconnaissance parfaite)
الْحَمْدُ لِلَّهِ رَبِّ الْعَالَمِينَ
🙏 Geste : Mains jointes en signe de gratitude (merci infini)
💡 Astuce : "Hamd" = "Louange" → comme "hommage" en français !
📚 Source : Quranic Arabic Corpus — racine ح-م-د (vérifiée dans v.2)
ع ل م الْعَالَمِينَ 🌍 "Al-'âlamîna" = "Les mondes" (toutes les créatures)
رَبِّ الْعَالَمِينَ
🌍 Geste : Bras qui font le tour de la terre (tous les mondes)
💡 Astuce : "'Âlamîn" = "Mondes" → comme "univers" mais en arabe !
📚 Source : Tafsir Ibn Kathir — tous les mondes (humains, djinns, anges) (vérifié dans v.2)
ر ب ب رَبِّ 👑 "Rabbi" = "Seigneur" (Celui qui crée, nourrit, éduque)
لِلَّهِ رَبِّ الْعَالَمِينَ
👑 Geste : Main ouverte vers le ciel (reconnaissance du Roi des mondes)
💡 Astuce : "Rabbi" = "Mon Seigneur" → comme "Roi" avec une couronne 👑 !
📚 Source : Quranic Arabic Corpus — racine ر-ب-ب (vérifiée dans v.2)
