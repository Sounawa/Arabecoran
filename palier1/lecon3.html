🌱 Leçon 3 : An-Nas 114:1-6

⏱️ 8 min • Sourate : An-Nas (114:1-6) • 📜 Lecture Warsh selon Nafi'
🕌 Lecture Warsh selon Nafi' : Cette sourate est identique dans les deux lectures principales (Hafs et Warsh).
✅ Texte vérifié : King Fahd Quran Complex (Édition Warsh officielle 1445 H)

قُلْ أَعُوذُ بِرَبِّ النَّاسِ • مَلِكِ النَّاسِ • إِلَٰهِ النَّاسِ • مِن شَرِّ الْوَسْوَاسِ الْخَنَّاسِ • الَّذِي يُوَسْوِسُ فِي صُدُورِ النَّاسِ • مِنَ الْجِنَّةِ وَالنَّاسِ

📜 ÉDITION WARSH SELON NAFI' • POLICE SHEHERAZADE NEW
🌙 Note éthique (Expert Coran) :
"An-Nas" = les gens. Cette sourate enseigne à chercher refuge auprès d'Allah contre le chuchoteur (Shaytan) qui inspire le mal dans les poitrines.
✅ Source : Tafsir Ibn Kathir, vol.8 p.528 — "Protection contre les chuchotements du diable"

🔑 Les 8 nouvelles racines (total : 18/50)

Racine arabe Mot dans le verset 🖐️ Geste 💡 Astuce + Source vérifiée
ن س س النَّاسِ 👥 "An-nâs" = "Les gens, l'humanité"
بِرَبِّ النَّاسِ
👥 Geste : Bras grands ouverts (comme pour englober toute l'humanité)
💡 Astuce : "Nâs" ressemble à "nous" → "nous tous, l'humanité" !
📚 Source : Quranic Arabic Corpus — racine ن-س-س (vérifiée dans v.1,2,3)
م ل ك مَلِكِ 👑 "Maliki" = "Roi" (Celui qui détient le pouvoir absolu)
مَلِكِ النَّاسِ
👑 Geste : Main sur le cœur avec inclinaison respectueuse (comme devant un roi)
💡 Astuce : "Malik" = "Roi" → comme "Malik" dans les contes avec une couronne 👑 !
📚 Source : Tafsir Ibn Kathir — attribut divin (vérifié dans v.2)
إ ل ه إِلَٰهِ 🙏 "Ilâhi" = "Dieu" (Celui qui est adoré)
إِلَٰهِ النَّاسِ
🙏 Geste : Mains jointes en prière (comme pour adorer)
💡 Astuce : "Ilâh" = "Dieu" → comme "Allah" mais sans le "Al-" !
📚 Source : Quranic Arabic Corpus — racine إ-ل-ه (vérifiée dans v.3)
و س و س الْوَسْوَاسِ 👂 "Al-wasswâs" = "Celui qui chuchote" (le diable)
مِن شَرِّ الْوَسْوَاسِ الْخَنَّاسِ
👂 Geste : Main près de l'oreille (comme pour écouter un chuchotement)
💡 Astuce : "Wasswâs" = "Chuchotement" → comme "whisper" en anglais !
📚 Source : Tafsir Al-Jalalayn — le diable qui chuchote (vérifié dans v.4)
خ ن س الْخَنَّاسِ 🦎 "Al-khannâs" = "Celui qui se cache/recule"
الْوَسْوَاسِ الْخَنَّاسِ
🦎 Geste : Main qui se cache derrière le dos (comme un lézard qui fuit)
💡 Astuce : "Khannâs" = "Celui qui fuit" → comme un lézard 🦎 qui disparaît !
📚 Source : Tafsir Ibn Kathir — le diable qui fuit quand on se souvient d'Allah (vérifié dans v.4)
ص د ر صُدُورِ ❤️ "Sudûr" = "Poitrines" (siège des pensées et des émotions)
فِي صُدُورِ النَّاسِ
❤️ Geste : Main posée sur la poitrine (là où bat le cœur)
💡 Astuce : "Sudûr" = "Poitrines" → comme "sœur" qui protège le cœur ❤️ !
📚 Source : Quranic Arabic Corpus — racine ص-د-ر (vérifiée dans v.5)
ج ن ن الْجِنَّةِ 👻 "Al-jinnati" = "Les djinns" (créatures invisibles)
مِنَ الْجِنَّةِ وَالنَّاسِ
👻 Geste : Doigts qui dansent dans les airs (comme des esprits invisibles)
💡 Astuce : "Jinn" = "Esprit invisible" → comme un fantôme 👻 !
📚 Source : Tafsir Ibn Kathir — les djinns sont une création d'Allah (vérifié dans v.6)
و وَ 🔗 "Wa" = "Et" (conjonction de liaison)
الْجِنَّةِ وَالنَّاسِ
🔗 Geste : Mains qui se rejoignent (comme pour lier deux choses)
💡 Astuce : "Wa" = "Et" → le plus petit mot mais il relie tout !
📚 Source : Quranic Arabic Corpus — particule de liaison (vérifiée dans v.6)
