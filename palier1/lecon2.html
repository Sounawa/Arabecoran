🌱 Leçon 2 : Al-Falaq 113:1-5

⏱️ 8 min • Sourate : Al-Falaq (113:1-5) • 📜 Lecture Warsh selon Nafi'
🕌 Lecture Warsh selon Nafi' : Cette sourate est identique dans les deux lectures principales (Hafs et Warsh).
✅ Texte vérifié : King Fahd Quran Complex (Édition Warsh officielle 1445 H)

قُلْ أَعُوذُ بِرَبِّ الْفَلَقِ • مِن شَرِّ مَا خَلَقَ • وَمِن شَرِّ غَاسِقٍ إِذَا وَقَبَ • وَمِن شَرِّ النَّفَّاثَاتِ فِي الْعُقَدِ • وَمِن شَرِّ حَاسِدٍ إِذَا حَسَدَ

📜 ÉDITION WARSH SELON NAFI' • POLICE SHEHERAZADE NEW
🌙 Note éthique (Expert Coran) :
"Al-Falaq" = l'aube. Cette sourate enseigne à chercher refuge auprès d'Allah contre les maux visibles et invisibles.
✅ Source : Tafsir Ibn Kathir, vol.8 p.521 — "Protection contre le mal de toute création"

🔑 Les 6 nouvelles racines (total : 10/50)

Racine arabe Mot dans le verset 🖐️ Geste 💡 Astuce + Source vérifiée
ع و ذ أَعُوذُ 🛡️ "A'oudhou" = "Je cherche refuge"
قُلْ أَعُوذُ بِرَبِّ الْفَلَقِ
🛡️ Geste : Bras croisés sur la poitrine (comme un bouclier)
💡 Astuce : "A'oudhou" ressemble à "abri" → "Je cherche un abri !"
📚 Source : Quranic Arabic Corpus — racine ع-و-ذ (vérifiée dans v.1)
ر ب ب رَبِّ 👑 "Rabbi" = "Mon Seigneur" (Celui qui crée, nourrit, protège)
أَعُوذُ بِرَبِّ الْفَلَقِ
👑 Geste : Main ouverte vers le ciel (reconnaissance du Roi)
💡 Astuce : "Rabbi" = "Mon Seigneur" → comme "Roi" avec une couronne 👑
📚 Source : Tafsir Ibn Kathir — attribut divin (vérifié dans v.1)
ف ل ق الْفَلَقِ 🌅 "Al-falaq" = "L'aube" (première lumière du jour)
رَبِّ الْفَلَقِ
🌅 Geste : Bras qui s'ouvrent lentement vers le haut (comme le soleil levant)
💡 Astuce : "Falaq" = "Fente" → l'aube "fend" l'obscurité de la nuit !
📚 Source : Quranic Arabic Corpus — racine ف-ل-ق (vérifiée dans v.1)
ش ر ر شَرِّ ❌ "Sharr" = "Mal" (tout ce qui nuit)
مِن شَرِّ مَا خَلَقَ
❌ Geste : Main qui fait "non" (index et majeur en croix)
💡 Astuce : "Sharr" = "Mal" → comme "charrue" qui retourne la terre (mais ici c'est le mal !)
📚 Source : Quranic Arabic Corpus — racine ش-ر-ر (vérifiée dans v.2,3,4,5)
غ س ق غَاسِقٍ 🌙 "Ghâsiq" = "Obscurité profonde" (de la nuit)
وَمِن شَرِّ غَاسِقٍ إِذَا وَقَبَ
🌙 Geste : Main devant les yeux (comme pour se protéger de l'obscurité)
💡 Astuce : "Ghâsiq" = "Obscurité" → comme "gaze" qui cache la vue !
📚 Source : Tafsir Al-Jalalayn — obscurité de la nuit (vérifié dans v.3)
ن ف ث النَّفَّاثَاتِ 🌬️ "An-naffâthât" = "Celles qui soufflent" (sorcières sur les nœuds)
وَمِن شَرِّ النَّفَّاثَاتِ فِي الْعُقَدِ
🌬️ Geste : Souffler doucement sur ses mains (comme pour chasser le mal)
💡 Astuce : "Naffâthât" = "Celles qui soufflent" → comme le vent 🌬️ qui souffle !
📚 Source : Tafsir Ibn Kathir — sorcières qui soufflent sur les nœuds (vérifié dans v.4)
