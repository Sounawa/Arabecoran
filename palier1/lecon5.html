الرَّحْمَٰنِ الرَّحِيمِ • مَالِكِ يَوْمِ الدِّينِ

📜 ÉDITION WARSH SELON NAFI' • POLICE SHEHERAZADE NEW
🌙 Note éthique (Expert Coran) :
"Ar-Rahmân Ar-Rahîm" = Le Très Miséricordieux, Le Très Clément.
"Mâliki yawmi-d-dîn" = Maître du Jour du Jugement.
✅ Source : Tafsir Ibn Kathir, vol.1 p.45 — "Attributs divins de miséricorde et de souveraineté"

🔑 Les 5 racines (total : 30/50)

Racine arabe Mot dans le verset 🖐️ Geste 💡 Astuce + Source vérifiée
ر ح م الرَّحْمَٰنِ 💖 "Ar-Rahmân" = Le Très Miséricordieux
الرَّحْمَٰنِ الرَّحِيمِ
💖 Geste : Main sur le cœur avec un grand sourire (amour infini)
💡 Astuce : "Rahmân" = Miséricorde sans limite → comme un océan d'amour !
📚 Source : Quranic Arabic Corpus — racine ر-ح-م (vérifiée dans v.3)
ر ح م الرَّحِيمِ 🤗 "Ar-Rahîm" = Le Très Clément
الرَّحْمَٰنِ الرَّحِيمِ
🤗 Geste : Bras ouverts pour faire un câlin (douceur constante)
💡 Astuce : "Rahîm" = Clémence continue → comme une pluie bienfaisante !
📚 Source : Tafsir Al-Jalalayn — miséricorde permanente (vérifié dans v.3)
م ل ك مَالِكِ 👑 "Mâliki" = Maître, Souverain (avec alif)
مَالِكِ يَوْمِ الدِّينِ
👑 Geste : Main sur le cœur avec inclinaison respectueuse (reconnaissance du Souverain)
💡 Astuce : "Mâlik" = Maître absolu → comme un roi 👑 sur son trône !
📚 Source : Édition Warsh selon Nafi' — écriture مَالِكِ (avec alif) (vérifié dans v.4)
ي و م يَوْمِ 🌅 "Yawmi" = Jour
مَالِكِ يَوْمِ الدِّينِ
🌅 Geste : Bras qui s'ouvrent vers le haut (lever du soleil)
💡 Astuce : "Yawm" = Jour → comme "jour" en français !
📚 Source : Quranic Arabic Corpus — racine ي-و-م (vérifiée dans v.4)
د ي ن الدِّينِ ⚖️ "Ad-dîni" = Jugement, Religion
يَوْمِ الدِّينِ
⚖️ Geste : Mains qui tiennent une balance (justice parfaite)
💡 Astuce : "Dîn" = Jugement → comme "destin" qui se décide !
📚 Source : Tafsir Ibn Kathir — Jour du Jugement (vérifié dans v.4)
